Titulo del Articulo

CSS es un lenguaje de hojas de estilo utilizado para darle presentación a sitios web creados con HTML, después de haber aprendido HTML lo siguiente que tienes que aprender es CSS, la definición anterior puede parecer un poco confusa pero te explico, CSS es el lenguaje que nos permite darle estilos, formas, tamaños, colores y posición a los elementos creados en un sitio web.
